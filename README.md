# fans-vibe-comparison

홍익대학교 2026년 1학기 소프트웨어공학 과제
"프로세스에 입각한 바이브코딩의 효과 논술" 비교 실험 저장소입니다.

동일한 주제/스택(Next.js + Supabase)으로 두 프로젝트를 운영하여,
프로세스 적용 여부에 따른 개발 결과를 비교합니다.
평소 자주 사용하는 JYP FANS 팬 커뮤니티 플랫폼을 구현하는 것을 공통 주제로 삼습니다.

---

## 프로젝트 구성

- [`fans-process-vibe`](./fans-process-vibe)
  - 소프트웨어공학 프로세스 기반으로 진행
  - GitHub Issues/Projects로 요구분석-설계-구현-테스트 관리
- [`fans-free-vibe`](./fans-free-vibe)
  - 자유형(비체계적) vibe coding 중심으로 빠르게 구현
  - 최소 문서화/경량 검증으로 진행

---

## 기술 스택

Next.js + Supabase

---

## 공통 구현 범위

두 프로젝트 모두 아래 범위를 동일하게 유지해 비교합니다.

### In Scope

- 커뮤니티 (게시글/댓글)
- 아티스트 공식 게시글/답글
- 이벤트 신청
- 멤버십 접근 제어

### Out of Scope

- 실시간 라이브/채팅 (WebRTC)
- 결제 및 글로벌 배송 풀플로우
- 오프라인 대기열/QR 현장 운영
